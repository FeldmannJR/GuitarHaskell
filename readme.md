# Guitar Haskell
## Trabalho de Tópicos Especiais em Computação IV(Prog. Funcional)

Guitar Hero feito em haskell usando gloss para os gráficos e proteaaudio para o audio.

Feito com @jvbelmonte

```

cabal install
cabal run
```
